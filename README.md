# Spice and Smile

A web-based food ordering platform built with PHP, JavaScript, and CSS. This project allows users to browse menus, place orders, and manage their accounts.

## Features
- User signup and login
- Menu browsing
- Cart and checkout
- Order history
- Rewards system

## How to Run
1. Place the project folder in your web server directory (e.g., `c:/wamp64/www/`).
2. Start your local server (WAMP, XAMPP, etc.).
3. Open `index.php` in your browser to access the homepage.

## Folder Structure
- PHP files: Backend logic and page rendering
- JS files: Frontend interactivity
- CSS files: Styling
- `images/`: Project images
